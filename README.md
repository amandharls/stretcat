# StretCat (Landing)

Landing page do app **StretCat**, feita em **HTML + CSS puro** (sem Tailwind, sem build).

## Como rodar

- **Opção 1 (mais simples)**: abra o `index.html` no navegador.
- **Opção 2 (recomendado)**: sirva com um servidor local (evita problemas de cache e caminhos).

Exemplo com Node:

```bash
npx serve .
```

Depois acesse o endereço exibido no terminal.
